How to open your app web:
-> Press "w"
-> Press "ctrl+c"
-> Run "npx expo install react-native-web react-dom @expo/metro-runtime || yarn expo add react-native-web react-dom @expo/metro-runtime"
-> Run "npm install || yarn"
-> Run "npx expo || yarn expo"
-> Press "w"
